# Paradigm
[![ko-fi](https://ko-fi.com/img/githubbutton_sm.svg)](https://ko-fi.com/L3L4Z8L38)

**Paradigm** is an essential server administration tool for Minecraft, designed to give you powerful control over server communication and automation. From scheduled announcements to advanced chat features, Paradigm provides everything you need to keep your community informed and engaged.

## Features

### Advanced Communication Tools
* **Automated Announcer:** Schedule messages to broadcast automatically at custom intervals. Control everything from the prefix to the header and footer.
* **Dynamic MOTD:** Create a fully custom Message of the Day that greets players when they join, with support for multiple lines and formatting tags.
* **Mentions System:** Allow players to ping each other with `@PlayerName` or notify everyone with `@everyone` (permission-based).
* **Group Chat:** Players can create their own private chat groups to communicate with friends.
* **Staff Chat:** A private channel for server staff to coordinate, complete with an optional boss bar indicator to show when you're in staff-chat mode.

### Powerful Customization
* **Full Formatting Support:** Utilize a wide range of tags to add **clickable links**, **runnable commands**, and **hover text** to your messages.
* **Hex Color Support:** Go beyond the default Minecraft colors and style your messages with any hex color code (`&#RRGGBB`).
* **Custom Commands:** Create your own server commands that can run multiple actions, send messages, or teleport players.

### Server Management
* **Restart Scheduler:** Schedule automated server restarts with configurable warning messages and countdowns to ensure your players are prepared.
* **Modular System:** Enable or disable any feature from the main config file to tailor the mod to your server's exact needs.

## Getting Started

### For NeoForge (via Sinytra Connector)
1.  Download and install **NeoForge**.
2.  Download **[Sinytra Connector](https://www.curseforge.com/minecraft/mc-mods/sinytra-connector)** and place it in your `mods` folder.
3.  Download **[Forgified Fabric API](https://www.curseforge.com/minecraft/mc-mods/forgified-fabric-api)** and place it in your `mods` folder.
4.  Download the latest **Fabric** version of Paradigm from the [**Releases Page**](https://modrinth.com/mod/paradigm).
5.  Place the Paradigm **Fabric** `.jar` file into your `mods` folder.

### For Fabric
1.  Download and install the **Fabric Loader**.
2.  Download the **[Fabric API](https://www.curseforge.com/minecraft/mc-mods/fabric-api)** and place it in your `mods` folder. **This is required.**
3.  Download the latest **Fabric** version of Paradigm from the [**Releases Page**](https://modrinth.com/mod/paradigm).
4.  Place the Paradigm `.jar` file into your server's `mods` folder.

### For Forge
1.  Download and install the recommended version of **Minecraft Forge** for your game version.
2.  Download the latest **Forge** version of Paradigm from the [**Releases Page**](https://modrinth.com/mod/paradigm).
3.  Place the Paradigm `.jar` file into your server's `mods` folder.

## Configuration

After running the server once with the mod, configuration files will be generated in your server's `config` folder.

### Settings
* All settings are located in a config folder: `config/Paradigm/...json`.
